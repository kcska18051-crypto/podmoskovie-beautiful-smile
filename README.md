# Интерактивный прототип «Красивая улыбка»

Прототип собран для интеграции внутрь действующего сайта Центра стоматологии «Подмосковье». Шапка и подвал обозначены как неизменяемая оболочка сайта; основная работа сосредоточена внутри страницы.

## Просмотр

Из папки проекта запустить:

```powershell
node server.mjs
```

Затем открыть `http://127.0.0.1:4173`.

## Что работает

- карточки ситуаций с раскрытием ответа;
- форма записи в модальном окне и отдельный консультационный блок;
- интерактивное сравнение «до/после»;
- мобильная карусель команды и общая карусель врачей;
- FAQ-аккордеон;
- лёгкое появление блоков с учётом настройки `prefers-reduced-motion`;
- адаптация под десктоп, планшет и мобильный экран.

## Перед переносом на сайт

- заменить демонстрационные изображения клинических случаев на утверждённые кейсы трёх масштабов;
- подключить формы к действующему обработчику сайта;
- использовать существующие компоненты шапки, подвала и рейтингов без копирования их разметки из прототипа;
- проверить актуальность врачей, рейтингов и количества отзывов в CMS;
- при необходимости собрать общую групповую композицию врачей из исходников без белых границ между портретами.

Автоматические проверки запускаются командой `node --test tests/*.test.mjs`.
